1. Project Overview

Three layers, one narrative: reproduce the paper, expose the constrained bottleneck, then show why the improved track is meaningfully better.

Baseline work

CellViT PanNuke baseline from the paper, used as the reference for mPQ and F1 detection.

Reproduced work

Constrained Assignment 2 run with frozen encoder and limited budget. It is intentionally weak so the gap is visible and honest.

Improved work

Unfrozen encoder, class-weighted cross entropy, AMP, and longer training to make the model better adapt to histopathology.

Key takeaways

Fine-tuning mattersUnfreezing the encoder produces a materially stronger representation for nuclei morphology.
Rare classes are hardInflammatory and Dead remain the lowest-performing categories across folds.
Generalization is realMoNuSeg performance suggests the representation transfers outside PanNuke.

4. Improvements & Contributions

The improved version changes the learning dynamics rather than simply making the run longer. The chart below shows the exact fold-level behavior.

  • Unfrozen encoderAllows end-to-end adaptation, which is especially important for nuclei textures and stain variability.
  • Class-weighted lossHelps the model pay attention to rare classes instead of overfitting dominant cell types.
  • AMPReduces memory pressure and makes the longer run practical on one GPU.
  • Longer training budgetImproved run uses 30 epochs versus 10 in the constrained setting.
Quantified impact
+6.90 mPQ
Improvement from 7.32% to 14.22% on PanNuke, with F1 detection improving from 22.86% to 31.49%.
